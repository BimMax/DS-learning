**Цель проекта:**

Построить модель предсказывающую уйдет клиент банка или нет. Метрика качества F1

**Описание проекта:**

Из банка начали уходить клиенты. Маркетологи подсчитали, что сохранить текущих клиентов дешевле, чем привлекать новых. Для этого нужно уметь выделять клиентов, которые вероятнее всего покинут банк в ближайшее время, чтобы провести индивидуальную работу с такими клиентами. 

В нашем распоряжении есть сторические данные о поведении клиентов и расторжении договоров с банком.
